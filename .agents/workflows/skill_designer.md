# 🎨 [스킬] 디자이너: 프리미엄 시스템 디자인 

이 스킬은 **디자이너 에이전트** 가 화면이나 컴포넌트 외형을 설계할 때 따르는 규칙입니다.

## 📝 디자인 원칙 워크플로우:
1. **컬러 팔레트 확정**:
   - 촌스럽고 쨍한 원색(#FF0000 등)은 피하고, 세련된 그라데이션과 부드러운 톤을 조합합니다.
   - 다크모드를 위해 깊은 어두운 톤의 배경(예: `#121212`)과 시인성 높은 텍스트를 고려합니다.
2. **동적 애니메이션 (Micro-interactions)**:
   - 버튼 호버(hover) 시 자연스러운 전환(`transition`) 효과를 필수로 적용하여 생동감을 줍니다.
   - 주요 요소가 나타날 때 바운스(Bounce)나 페이드-인(Fade-in) 모션을 도입하세요.
3. **모바일 최적화 (Mobile First)**:
   - 좁은 모바일 화면에서도 텍스트나 버튼이 지저분하게 줄바꿈되지 않도록 깔끔하게 설계합니다.

> **💡 중요**: 구현 시, 재사용 가능한 스타일 토큰이나 CSS 시스템을 활용하여 디자인의 일관성을 유지하세요.
