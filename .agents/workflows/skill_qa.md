# 🕵️‍♂️ [스킬] QA: 무결성 검증 및 엣지 케이스 테스트 

이 스킬은 **QA 에이전트** 가 버그를 체크하고 UI의 완성도를 테스트할 때 참고하는 매뉴얼입니다.

## 📝 테스트 체크리스트 및 워크플로우:
1. **크로스 디바이스 뷰포트 확인**:
   - 모바일 해상도(375px ~ 430px 등)에서 요소가 밖으로 튀어나오거나, 여백이 이상한 곳은 없는지 확인합니다.
2. **에러 핸들링 (Error Boundary)**:
   - 번호를 저장하는 과정에서 실패했을 경우 (에러 상황), 유저에게 안내 창이 잘 뜨는지 점검합니다. 
   - 브라우저 콘솔에 예상치 못한 Warning이나 Error 스택이 찍히지 않는지 찾아냅니다.
3. **메타 정보 및 공유 기능**:
   - SEO를 위한 오픈 그래프(`og:image`, `og:title`)가 유효하게 적용되었는지 검증합니다.
4. **비즈니스 로직 테스트 무결성**:
   - 생성된 로또 숫자가 1~45 사이를 벗어나지는 않았는지, 중복이 발생하지 않았는지 가혹한 환경에서도 견고한지 테스트합니다.

> **💡 원칙**: 문제를 리포팅할 때는 발견된 상황(재현 경로), 현재 문제점, 기대되는 결과 모두를 완벽한 한글 문장으로 기재하세요.
